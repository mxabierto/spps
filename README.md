# MIIPPS
Dashboard SPPS-DataLab

# Setup instructions
## Dependencies 
1. Postgre SQL database >= 9.4 
2. NodeJS v6 or later 

## Web app
1. `cd spps/`
2. `npm install`
3. `cd public/ && bower install`

# Running the app
1. `cd spps/ && ./bin/www`
2. Open `http://127.0.0.1:4444/` in your web browser
